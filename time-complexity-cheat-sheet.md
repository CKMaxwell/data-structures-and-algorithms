# time complexity cheat sheet

| Search Algorithm | time complexity function(F(N)) | Big-O(O(N)) |
| ---------------- | ------------------------------ | ----------- |
| Linear Search    | F(N) = N                       | O(N)        |
| Binary Search    | F(N) = log2(N)                 | O(log N)    |

## Other Common Algorithm

- Counter
- Pointer
- sliding window
